В этой директории проект мигалка "Вильгельм Тель" светодиодом: "та-да-дам, та-да-дам, та-да-дам, там, там". Ритм задается одной строкой по принципу: нота = два параметра {1,2} , где 1 = длительность свечения СВД, 2 = длительность гашения СВД. Длительности вырабатываются с помощью прерываний от таймера  TC0.
